# 黄金走势工作台授权接入说明

## 当前接入参数

- 授权中心：`https://license-center.example.com`
- 黄金产品 ID：`gold-trend-desk`
- 产品名称建议：`黄金走势工作台`
- 客户端公钥：`license-public.pem`

公开版默认以本地演示模式启动，不强制连接授权中心。正式商业部署时，再通过环境变量开启授权。

## 在授权中心新增产品

1. 打开你的授权中心后台，例如 `https://license-center.example.com/`。
2. 使用顶级管理员账号登录。只有顶级管理员或具备 `products:manage` 权限的账号能新增产品。
3. 进入左侧 `产品系统`。
4. 点击 `新增产品`。
5. 按下面内容填写：
   - 产品 ID：`gold-trend-desk`
   - 名称：`黄金走势工作台`
   - 当前版本：`1.0.1`
   - 状态：`启用`
   - 备注：可写 `黄金行情与走势预测工作台`
6. 点击 `保存产品`。

产品 ID 必须和本网站后端的 `PRODUCT_ID` 完全一致，否则许可证会被判定为不属于当前产品。

## 为黄金工作台生成许可证

1. 进入左侧 `许可证`。
2. 点击 `新建许可证`。
3. 产品选择 `黄金走势工作台 (gold-trend-desk)`。
4. 填写客户名称、套餐名称、生效时间和到期时间。
5. 设置最大激活设备数。个人使用通常填 `1`，团队交付可按实际设备数填写。
6. 设置在线租约时长和断网宽限：
   - 在线租约时长：建议 `6` 小时。
   - 断网宽限：建议 `24` 小时。
7. 授权功能可以全选。黄金工作台当前不细分功能权限，但保留字段便于以后扩展。
8. 点击 `保存`。
9. 系统会弹出许可证密钥。这个密钥只显示一次，需要立即保存并交给客户或填写到黄金工作台激活页。

## 激活黄金工作台

1. 启动黄金工作台：

```powershell
.\start.ps1
```

2. 打开 `http://localhost:4173`。
3. 在激活页输入授权中心生成的许可证密钥。
4. 点击 `联网激活`。

激活成功后，本机会保存安装编号、设备指纹和中央授权租约。后续打开会自动验证授权；授权过期、吊销或设备释放后，网站会重新进入激活状态。

## 临时开发模式

如果只是开发页面，不想被授权拦截，可以这样启动：

```powershell
$env:LICENSE_REQUIRED="false"
.\start.ps1
```

如需模拟商业部署，可以显式开启授权：

```powershell
$env:LICENSE_REQUIRED="true"
$env:LICENSE_SERVER_URL="https://license-center.example.com"
$env:LICENSE_PUBLIC_KEY_PATH="C:\path\to\license-public.pem"
.\start.ps1
```

## 云端更新发布

1. 在授权中心后台进入 `云端更新`。
2. 选择产品 `黄金走势工作台 (gold-trend-desk)`。
3. 填写版本号、通道和发布说明：
   - 版本号：必须高于当前网站版本，例如 `1.0.1` 之后发布 `1.0.2`
   - 通道：正式客户用 `stable`，测试客户可用 `beta`
   - 包类型：服务器部署包使用 `server-archive`
4. 上传客户部署包。
5. 保存后确认后台已经生成 SHA256、签名、文件大小和发布时间。
6. 把状态设为 `active` 后，已授权的网站才能在设置页 `云端更新` 中检查和下载。

更新包只允许已授权客户检查和下载。当前黄金工作台默认只支持“下载并验签”，不会自动覆盖服务器文件；正式上线仍建议按部署流程先备份，再替换文件并重启服务。
